------------------------------------------------------------
5. Why Certificate Does Not Appear in Wireshark
------------------------------------------------------------

In TLS 1.3, all handshake messages after the Server Hello are encrypted,
including:

  - Certificate
  - CertificateVerify
  - Finished

Because of this, the server certificate does not appear in plaintext
in Wireshark. This behavior is expected and indicates a modern, secure
TLS 1.3 configuration used by Facebook and other large providers.

The Certificate message is encrypted by design in TLS 1.3, demonstrating
that the server implements secure TLS 1.3 handshake practices.
